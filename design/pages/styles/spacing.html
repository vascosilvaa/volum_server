Espaçamentos

Para uma melhor organização e sintonia gráfica, é necessário estabelecer padrões de espaçamento para todos os componentes.

Padding
padding: 16px

O padding: 16px é utlizado para o espaçamento interior de todos os componentes.

Margin
margin: 16px

A margin: 16px é utlizada para o espaçamento entre todos os componentes.
